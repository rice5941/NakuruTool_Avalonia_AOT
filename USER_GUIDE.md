# NakuruTool ユーザーガイド

このガイドは、NakuruTool を初めて使う方向けの基本操作ガイドです。
## アプリでできること

- osu! の beatmap を一覧で確認できます。
- 条件を組み合わせて beatmap を絞り込めます。
- 絞り込み結果から collection.db にコレクションを生成できます。
- フィルタ条件をプリセットとして保存し、再利用できます。
- 既存コレクションをエクスポートし、別ファイルからインポートできます。
- オーディオパネルで譜面のプレビュー再生、曲送り、シーク、音量調整ができます。
- Language/言語、テーマ、再生動作、表示方法を設定できます。

## 最初にやること

初回起動時は 設定 画面が表示されます。

1. osu!フォルダパス の項目で osu!.exeを選択 を押し、osu!.exe を選びます。
2. 必要に応じて Language/言語 と テーマ を調整します。
3. 選択時自動再生 と Unicode表示を優先 を好みに合わせて切り替えます。
4. 読み込みが終わったら beatmap一覧 に戻って操作を始めます。

まず osu! フォルダを設定しないと、beatmap一覧、コレクション生成、インポートの一部機能を正しく使えません。

## 画面構成

1. 左側のサイドバーで beatmap一覧、インポート/エクスポート、ライセンス、設定 を切り替えます。
2. 上部のメニューボタンでサイドバーを畳むことができます。1920 x 1080 より低い解像度の画面では、畳んで使うと操作しやすくなります。
3. beatmap一覧 では、左側でフィルタ、右側で譜面一覧を確認します。
4. インポート/エクスポート では、左側でファイルやコレクションを選び、右側で内容を確認します。
5. 起動直後は、osu!.db、collection.db、scores.db の読み込みが完了するまで待ってから操作してください。

## beatmap一覧の見方

1. 一覧上部の map数 で読み込んだ総数、絞込後 で現在の対象件数を確認します。
2. 難易度値 列では、キー数と ★ の難易度を確認します。
3. 曲名 列では、曲名、難易度、アーティスト、作成者をまとめて確認します。
4. BPM と LN率 を見て、譜面傾向を把握します。
5. スコア、履歴、プレイ済み を見て、過去のプレイ状況を確認します。
6. 必要に応じて v1(default) と v2、通常 と HT と DT/NC を切り替えてスコア表示を切り替えることができます。
7. 一覧下部の左右ボタン、ページ番号、件数切替でページを移動します。

譜面を選ぶと、選択時自動再生 がオンのときは音声プレビューが始まります。

## オーディオパネルの使い方

1. beatmap一覧 の上部にあるオーディオパネルを使います。
2. 譜面を選ぶと、タイトル、難易度、アーティスト、作成者、背景画像が更新されます。
3. 中央の 再生/一時停止 ボタンで現在の譜面を再生または一時停止します。
4. 前の曲 と 次の曲 を使うと、現在の絞り込み結果の順番に沿って譜面を移動できます。
5. シャッフル をオンにすると、絞り込み結果の中からランダムに再生できます。
6. リピート は 押すたびに なし → 全体リピート → 1曲リピート の順に切り替わります。
7. シークバーを動かすと再生位置を調整でき、右側のスライダーで音量を変更できます。

前の曲 は、3秒以上再生している場合は前の譜面へ移動せず、現在の曲の先頭に戻ります。

## フィルタの使い方

1. beatmap一覧 の左側にある フィルタ条件 を確認します。
2. 条件を追加 を押し、対象 と 比較条件 を選びます。
3. 値、最小値、最大値、日付、コレクションなど、表示された入力欄に条件を入れます。
4. 条件を増やしたいときは、もう一度 条件を追加 を押します。
5. 複数の条件はすべて AND 条件で組み合わされます。
6. 不要な条件は各条件の削除ボタンで外し、全部消したいときは すべてクリア を使います。
7. 保存済み条件を使うときは プリセット を選び、解除したいときは (なし) を選びます。

## プリセットの使い方

1. フィルタ条件 の プリセット 欄の右にあるボタンから プリセット編集を開く を開きます。
2. 追加 を押して新しいプリセットを作ります。
3. プリセット名を入力し、必要な条件を追加します。
4. 内容が決まったら 保存 を押します。
5. 使わなくなったプリセットは 削除 で消せます。
6. 複数のプリセットからまとめてコレクションを作りたいときは 一括生成 を使います。

保存したプリセットは、アプリ本体と同じ場所の presets フォルダに保存されます。

## コレクション生成

1. beatmap一覧 で必要なフィルタ条件を設定します。
2. 左下の コレクション生成 で、コレクション名を入力してください に作成名を入れます。
3. 入力欄の右にある保存アイコンのボタンを押して生成を開始します。
4. 対象が 10000 件を超える場合は確認が表示されるので、問題なければ 生成する を押します。
5. 処理をやめるときは キャンセル を押します。
6. 生成中と生成後は、下に表示されるステータスメッセージと進捗バーで状況を確認します。

同名コレクションを生成すると、既存の同名コレクションが上書きされます。残したいコレクションがある場合は別の名前を使ってください。

## インポート/エクスポート

### エクスポート

1. インポート/エクスポート タブを開きます。
2. 上段の エクスポート で対象コレクションにチェックを入れます。
3. 必要に応じて リロード、すべて選択、すべて解除 を使います。
4. エクスポート実行 を押します。
5. 完了したファイルは exports フォルダに保存されます。

### インポート

1. 取り込みたいファイルを imports フォルダに入れます。
2. インポート/エクスポート タブの下段にある インポート を開きます。
3. ファイル再読込 を押して一覧を更新します。
4. 取り込みたいファイルにチェックを入れます。
5. 必要に応じて すべて選択、すべて解除 を使います。
6. 右側の一覧で map数 や 存在 を確認します。
7. インポート実行 を押して collection.db に反映します。

インポート用ファイルはアプリ本体と同じ場所の imports フォルダ、エクスポート結果はアプリ本体と同じ場所の exports フォルダを使います。

## 設定

1. 設定 タブを開きます。
2. Language/言語 で表示言語を選びます。
3. テーマ で見た目を切り替えます。
4. osu!フォルダパス で現在の設定を確認し、必要なら osu!.exeを選択 で変更します。
5. 選択時自動再生 をオンにすると、譜面選択時にプレビュー再生しやすくなります。
6. Unicode表示を優先 をオンにすると、曲名やアーティスト名を Unicode で表示できます。日本語や韓国語の表記を優先したい場合に有効です。

## ライセンス

1. ライセンス タブを開きます。
2. アプリで使用しているオープンソースライセンスの一覧を確認します。
3. 詳しく確認したい項目があれば 全文を表示 を開きます。

## 復旧方法

- 本アプリの起動時にアプリ本体と同じ場所の backups フォルダにcollection.dbのバックアップを作成しています。もしcollection.dbが破損した場合、backupsフォルダからcollection.dbを上書きしてください。
